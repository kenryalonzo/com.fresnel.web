# Installation de react-toastify

Le package `react-toastify` est déjà ajouté au `package.json` mais n'est pas encore installé.

## Méthode 1 : Via le terminal

Ouvrez un terminal dans le dossier du projet et exécutez :

```bash
npm install
```

ou spécifiquement :

```bash
npm install react-toastify
```

## Méthode 2 : Via le script batch

Double-cliquez sur le fichier `install-toastify.bat` dans le dossier racine du projet.

## Vérification

Après l'installation, redémarrez le serveur de développement :

```bash
npm run dev
```

L'erreur devrait être résolue !
